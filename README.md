# 🏠 House Price Prediction

## 📌 Overview
This project predicts house prices using machine learning techniques on real-world data.

## ⚙️ Tech Stack
- Python
- Pandas
- Scikit-learn

## 🤖 Models Used
- Linear Regression
- Random Forest Regressor

## 📊 Results
- Linear Regression R²: 0.37
- Random Forest R²: 0.93

## 🔍 Key Insights
- Total_Sq.ft is the most important feature
- Feature selection improved performance significantly
- Simpler models can perform nearly as well as complex ones

## 🚀 Learnings
- Data cleaning and preprocessing
- Feature engineering
- Model evaluation using R² score
- Feature importance analysis
